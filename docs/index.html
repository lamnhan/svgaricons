SVG-r-ICONS!

Another way to use SVG icons with SCSS. It's "SUGAR ICONS", baby!

Source code: https://github.com/lamnhan/svgricons


Why SVG, why not Font icons?

Consider the graphical design background, I personally refer the SVG icons over the Font icons because with SVG I can easily build my own icon packs (no matter if a website has 1 icon or hundreds), as well as the ability to customize the SVG image ifself.

1. Install

npm i @lamnhan/svgricons

2. Define the theme

:root {
  // colors
  --color-primary: #3880ff;
  --color-secondary: #eb445a;
  --color-success: #2dd36f;
  // recolors
  --recolor-primary: brightness(0) saturate(100%) invert(37%) sepia(73%) saturate(1013%) hue-rotate(193deg) brightness(106%) contrast(104%);
  --recolor-secondary: brightness(0) saturate(100%) invert(45%) sepia(84%) saturate(2892%) hue-rotate(324deg) brightness(91%) contrast(102%);
  --recolor-success: brightness(0) saturate(100%) invert(90%) sepia(90%) saturate(2470%) hue-rotate(65deg) brightness(90%) contrast(82%);
}

P/S: Use this tool for recoloring: https://angel-rs.github.io/css-color-filter-generator/

3. Register icons

@import '@lamnhan/svgricons';
// Option 1: simple icons
@include svgricons(
  (
    github: '/icons/github.svg',
    github2: '/icons/github2.svg', // variant of 'github'
    google: '/icons/google.svg',
    google2: '/icons/google2.svg', // variant of 'google'
    apple: '/icons/apple.svg',
    apple2: '/icons/apple2.svg', // variant of 'apple'
  )
);
// Ontion 2: icons with custom sizes, colors
@include svgricons(
  (/* ... */),
  (
    sizes: (10, 16, 32, 48, 64),
    colors: ('primary', 'secondary', 'success'),
  )
);

4. Usage

Default

<i class="icon github"></i>
<i class="icon google"></i>
<i class="icon apple"></i>

Variants

<i class="icon github2"></i>
<i class="icon google2"></i>
<i class="icon apple2"></i>

Sizes

<i class="icon apple size-10"></i>
<i class="icon apple size-16"></i>
<i class="icon apple size-32"></i>
<i class="icon apple size-48"></i>
<i class="icon apple size-64"></i>

Colors

<i class="icon github size-48 color-primary"></i>
<i class="icon google size-48 color-secondary"></i>
<i class="icon apple size-48 color-success"></i>

Hovered (colors)

<i class="icon google size-48 hovered-primary"></i>
<i class="icon google size-48 color-primary hovered-secondary"></i>
<button class="icon-hovered-success">
  <i class="icon google"></i>
  <span>Hover me</span>
</button>

Hovered (different sources)

<i class="icon apple size-48 hovered-apple2"></i>
<i class="icon apple2 size-48 hovered-apple"></i>
<button class="icon-hovered-apple2">
  <i class="icon apple"></i>
  <span>Hover me</span>
</button>
